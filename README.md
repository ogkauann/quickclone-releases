## O que é o QuickClone?

Ferramenta para clonar servidores Discord completos com canais, roles, permissões, emojis e mensagens.

- **Versão Gratuita**: Canais, categorias, roles e permissões
- **Versão Pro**: Emojis, stickers, mensagens, agendamento, templates e mais
